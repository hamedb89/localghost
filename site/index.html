Localghost

A tiny Node.js CLI for local HTTPS domains in app repos. Use one `.localghost` contract for Caddy, `/etc/hosts`, Vite `allowedHosts`, and clean browser-facing URLs.

View on GitHub Install from npm 
yarn add -D @hamedb89/localghost

yarn localghost init --write-scripts
yarn localghost doctor
yarn localghost:setup
yarn localghost:proxy

One small contract for local dev

.localghost

Map friendly `.localhost` names to app ports in a plain text file that humans and agents can inspect.

Caddy and hosts

Generate local HTTPS proxy config and update only Localghost's managed block in `/etc/hosts`.

Vite-safe defaults

Set explicit `server.allowedHosts` entries without opening the broad `allowedHosts: true` shortcut.

Built to be boring where it matters

CI before publish

GitHub Actions runs typecheck, build, site build, and npm package dry-run on Node 20 and 22.

Small dependency surface

Runtime dependencies are limited to `commander` and `execa`. Vite stays an optional peer for projects that use the plugin.

No surprise installs

No postinstall scripts, hidden Homebrew commands, or full hosts-file rewrites. Localghost changes only its managed block.

MIT licensed. Built for friendly local development without surprise spells.
